## Quick start

1.  **Development.**

    For localhost

    ```shell
    yarn develop
    ```

    For specified host (specify host location in package.json)

    ```shell
    yarn start
    ```

2.  **Other commands.**

    Build production

    ```shell
    yarn build
    ```

    Serve production

    ```shell
    yarn serve
    ```

    Test production build (build and serve)

    ```shell
    yarn test
    ```
